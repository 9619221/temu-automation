# v0.1.83 更新说明书

## 版本信息

- 版本号：`v0.1.83`
- 更新时间：`2026-04-24`
- 适用对象：当前正在使用 Temu 自动化运营工具的桌面端用户

## 一、如何更新

你可以用下面两种方式更新到 `v0.1.83`：

### 方式 1：应用内更新

1. 打开软件
2. 进入 `设置`
3. 点击 `检查更新`
4. 如检测到 `v0.1.83`，点击 `下载更新`
5. 下载完成后点击 `立即安装更新`

### 方式 2：手动安装覆盖更新

如果应用内没有及时拉到更新，也可以直接下载安装包覆盖安装：

- 安装包：`temu-automation-setup-0.1.83.exe`

覆盖安装不会影响已有账号、历史数据和常规使用配置。

## 二、本次更新重点

本次更新主要围绕 `AI 生图 GPT 版` 做了功能增强和稳定性修复。

### 1. 新增：单张总览图识别多个商品并做组合装生图

现在可以先上传 `1 张包含多个商品 / 配件的总览图`，系统会自动识别图中可售卖组件，并给每个组件打上序号。

你只需要：

1. 上传总览图
2. 点击 `识别并编号`
3. 勾选需要组合销售的序号
4. 再进入分析和生图

当你选择 `2 个及以上序号` 时，系统会自动按 `组合装 bundle` 逻辑继续处理。

### 2. 新增：裁剪素材可视化

之前系统虽然会在内部处理组合装素材，但页面上看不到实际送入分析和生图的裁剪结果，容易让人误以为“没有裁剪”。

本次更新后：

- 页面会直接显示 `实际送入分析 / 生图的裁剪素材`
- 你可以先确认裁剪结果对不对，再继续分析和生成
- 系统后续会优先使用这批裁剪后的素材，而不是继续拿整张总览图硬做

### 3. 修复：桌面端白屏问题

修复了部分情况下桌面端进入相关页面时出现白屏的问题，提升了 `AI 生图 GPT 版` 的稳定性。

## 三、更新后的推荐使用流程

如果你要做组合装图片，推荐按下面流程使用：

1. 进入 `AI 生图 GPT 版`
2. 上传 `1 张` 多商品 / 多配件总览图
3. 点击 `识别并编号`
4. 选择需要组合的序号
5. 查看右侧 `实际送入分析/生图的裁剪素材`
6. 确认无误后点击 `开始 AI 分析`
7. 分析完成后点击 `开始生成`

## 四、使用规则说明

### 1. 选择 0 或 1 个序号

系统仍按普通单品流程处理，不会自动当作组合装。

### 2. 选择 2 个及以上序号

系统会自动按 `bundle / 组合装` 逻辑分析和生图。

### 3. 为什么要看裁剪素材预览

因为组合装效果是否准确，根本上取决于：

- 识别框是否准确
- 裁剪结果是否干净
- 送入分析的到底是不是你想要的组件

这次更新就是把这一步变成可见、可确认的流程。

## 五、适合哪些场景

这次更新特别适合下面几类商品：

- 多件工具套装
- 多配件组合包
- 单张包装图里包含多个独立售卖部件的商品
- 想先编号再自由组合出图的商品

## 六、常见问题

### 1. 识别后没有进入组合装

请先确认是否勾选了至少 `2` 个序号。  
如果只选了 `1` 个，系统仍会按单品处理。

### 2. 裁剪结果不准确怎么办

建议优先更换更清晰、更正面、背景更干净的总览图，再重新执行一次 `识别并编号`。

### 3. 更新后原有功能会受影响吗

不会。原来的普通单品分析、生图、历史记录等功能仍可正常使用。

## 七、建议

如果你当前做的是组合装商品，建议优先使用本次新增流程：

`上传总览图 -> 识别并编号 -> 选择组合 -> 确认裁剪结果 -> 分析 -> 生成`

这样能明显减少“整张图被当成单个商品”或“组合关系不准确”的问题。
