# v0.3.7 更新说明

## 这次更新

- 数据采集页新增浏览器采集助手安装引导。
- 默认安装入口指向本次 Release 的扩展 ZIP 文件，用户可以直接下载扩展文件。
- 设置页新增扩展文件下载链接、Chrome Web Store 链接、云端地址和 Token 配置。
- 采集页会展示扩展心跳、在线设备、待上报队列和 Hook 状态。
- 合入多店云监控采集扩展与云端 ingest 服务能力。

## 发布资产

除了桌面客户端更新文件，还需要在 GitHub Release 上传：

- `temu-collection-extension-v0.3.7.zip`

用户点击“下载扩展文件”后，会打开这个 Release 资产。
